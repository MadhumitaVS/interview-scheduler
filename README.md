# Interview Scheduler & Attendance Tracker (MVP)

## Overview
A front-end web application to schedule interviews, open join links, and mark attendance.  
This is a frontend-only project using mock JSON and localStorage for persistence.

## Tech Stack
- React + Vite
- JavaScript (ES6)
- CSS (custom styling)
- localStorage for data persistence

## Features
- Schedule interviews (Company, Role, Date, Time, Mode)
- Open interview join link
- Mark interview as attended or missed
- Search interviews by company, role, interviewer
- Reset data to defaults
- Data persists using localStorage
